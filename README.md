# Friends management

## Getting Started
 
### Prerequisites(if wanna run in local)
 
```
git
Ruby (version 2.5.0)
Rails (version 5.1.6)
```

**Clone the repo
 
```
git clone git@github.com:priyasahu/friends-management.git
```
 
**Install gems

```
bundle install
```
 
**Database migration

```
rails db:create
rails db:migrate
rake db:seed
```
 
**Now you may run a local server with
```
rails s
```
